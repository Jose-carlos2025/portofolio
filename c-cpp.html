Curiosidades e Dicas

  • C é uma das linguagens mais antigas ainda em uso e base para muitas outras, como C++, Java e Python.
  • O símbolo ++ em C++ significa "incremento", indicando evolução da linguagem C.
  • Ponteiros são um dos conceitos mais poderosos (e temidos) em C/C++.
  • Grandes sistemas operacionais, como Linux e Windows, têm partes escritas em C/C++.
